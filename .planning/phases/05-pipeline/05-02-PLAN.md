## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| shipping_tracker → TrackingMore API (HTTPS) | Outbound request carrying the API key in the Tracking-Api-Key header; response is untrusted input parsed in _handle |
| env (`.env`) → process | TRACKINGMORE_API_KEY enters the process; must never leave via logs or exceptions |
| parser output → registrar | tracking_number / carrier cross into the request body (already validated upstream by the parser) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03 | Information Disclosure | API key in logs/exceptions | mitigate | Key read from env only (D-05); `config.missing_api_key` logs no value; no log/exception in registrar.py or main.py interpolates `self._api_key` — enforced by `test_api_key_never_logged` and a source-grep acceptance criterion |
| T-05-04 | Information Disclosure | tracking_number / carrier in logs | mitigate | All registrar log lines and exception messages are structural only ("registrar.created", "rate-limit", "quota-exhausted"); WR-04 logs message_id + type only; enforced by `test_tracking_number_never_logged` |
| T-05-05 | Tampering | Malformed/non-JSON TrackingMore response | mitigate | Defensive `try resp.json() except → {}` (Pitfall 6); unknown meta.code falls through to return False (no persist) rather than crashing |
| T-05-06 | Denial of Service | Hung socket stalls cron job | mitigate | 10s httpx timeout (D-03) + single ~2s retry caps a dead endpoint at ~22s, then defers (D-02); httpx.Client closed in finally (Pitfall 4) |
| T-05-07 | Tampering | SSRF via tracking_number | accept | URL is the hardcoded `_BASE_URL` constant, never constructed from input — no SSRF surface (RESEARCH Security Domain) |
| T-05-SC | Tampering | httpx/respx supply chain | accept | Both audited `[OK]` in 05-RESEARCH.md Package Legitimacy Audit; httpx already in-tree; no `[ASSUMED]`/`[SUS]` packages — no human checkpoint required |
