<objective>
Permanently remove the maintainer's real personal email (`<maintainer-real-email-redacted>`)
from the git repository's full history and working tree, replacing it with the GitHub
noreply form (`9119417+Swissola@users.noreply.github.com`) already used in commit metadata,
then force-push the scrubbed history to the private `origin` remote so the repo can be made
public with no PII.

real email in any commit blocks release.

Output: Rewritten local history with the email scrubbed from all ~20 affected commits, a
recoverable pre-rewrite backup, refreshed stale hash references, and a force-pushed clean
`origin/main`.

This is an IRREVERSIBLE git history rewrite executed on the MAIN working tree (no worktree
isolation). Tasks are strictly ordered; do not reorder. Task 1 (backup) gates the rewrite,
and Task 5 (verification) gates the force-push (Task 7 — the point of no return).
</objective>
